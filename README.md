# Simple Data Cleaning Assignment

## Overview
This project demonstrates basic data cleaning techniques applied to a small dataset.
The goal was to prepare raw data for analysis by improving structure, consistency,
and usability.

## Tasks Performed
- Inspected the dataset for inconsistencies
- Corrected data types
- Handled or removed invalid entries
- Verified the final cleaned dataset

## Tools Used
- Microsoft Excel (data cleaning, formulas, basic analysis)

## Outcome
The final dataset is clean, structured, and ready for analysis or visualization.
This project reflects foundational data-cleaning skills used in real-world analytics tasks.
